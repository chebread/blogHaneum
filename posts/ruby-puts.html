Ruby 에서 출력하는 방법
2021년 8월 21일 툐요일 🌦

Ruby 에서는 출력하기 위해 puts 메소드를 사용해요

puts 메소드를 사용하여 Hello, world! 를 출력하려면, 이런식으로 코드를 작성하면 되요! 😉

                    puts ("Hello, world!")
                    puts "Hello world!" # 괄호 없이 사용할 수도 있어요

또한, print 메소드로도 출력이 가능해요

                    print ("Hello, world!")
                    print "Hello, world!" # print 메소드 또한 괄호 없이 사용 가능해요
