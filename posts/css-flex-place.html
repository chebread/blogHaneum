CSS Flex의 배치 방법
2021년 8월 4일 수요일 ⛅️

flex-item 들의 배치할 방향을 정하는 방법

배치할 방향은 flex-direction: ...로 설정할 수 있어요 😉

row (기본 속성)

flex-item 들을 가로로 배치해요

row-reverse

flex-item 들을 가로방향 역순으로 배치해요

column

flex-item 들을 세로방향으로 배치해요

column-reverse

flex-item 들을 세로방향 역순으로 배치해요

flex-item 들의 줄넘김 처리 설정방법

줄넘김 처리는 flex-wrap: ... 으로 설정해요

nowrap (기본 속성)

아이템들이 밑으로 떨어지지 않고 버티는 속성이에요

wrap

아이템들이 밑으로 떨어지게 하는 속성이에요

wrap-reverse

요소들이 떨어지지만, 위로 떨어지는 속성이에요

flex-direction 과 flex-wrap 을 한꺼번에 지정하는 방법

flex-flow: (flex-direction 속성) (flex-wrap 속성)으로 한꺼번에 지정할 수 있어요 🤩
