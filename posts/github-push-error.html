Github push 오류 해결 방법
2021년 7월 28일 수요일 🌥

만약, Github 로 push 를 시도할때, remote: ...오류가 난다면 이 해결 방법을 참고하면 되요 😎

해결 방법은, Mac 기준이에요


  • Keychain에 github을 검색하고, Internet으로 된 속성을 삭제해요

    •       삭제하는 방법은, 요소에 ctrl을 누르면서 클릭하면 작은 팝업으로 제거가 나와요!

  • Github의 설정 → 개발자 설정 → token 발급으로 들어가서 토큰을 repo 를 전체 선택후 token을 복사해요
  • git push 를 입력하고, 그다음 나오는 id 입력 상자에 terminal에서 사용자 이름을 입력해주고, 비밀번호에 token을 입력해줘요
  • token은 재발급이 안되니 안전한 곳에 보관해줘요
  • 이제 기존에 push 했던거처럼 동일하게 push 가 가능해요
  • 여기서 알아두어야 할점은, github token을 발급하였으면 token이 유효한 지정한 일 수가 있을 것인데,
    만약 그 일 수가 지나면 token을 다시 발급을 해야해요 (다시 발급할때는 ①, ② 번은 안해도되요!)
