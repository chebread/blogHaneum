Github push 오류 해결 방법
2021년 7월 28일 수요일 🌥

오늘 Github에 push 하려고 하니, remote: (영어..) 이런식으로 결과가 나오더라고요.. 🤔

그래서 열심히 검색한 결과, 해결 방법을 찾았어요! 🤩

(해결 방법은 Mac 기준이에요!)

1. Keychain에 github을 검색하고, Internet으로 된 속성을 삭제해요

      삭제하는 방법은, 요소에 ctrl을 누르면서 클릭하면 '제거'가 나와요!

2. git push을 시도해보고, push가 정상적으로 작동되면 성공이에요!
