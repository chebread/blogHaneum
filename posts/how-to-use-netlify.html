Netlify 사용 방법
2021년 8월 21일 토요일 ✨

Netlify 사용 방법에 대해 알아볼게요!

기본으로 Netlify 회원가입이 진행되어 있어야 해요

내가 만든 페이지를 Netlify에 배포 하는 방법

  • netlify.toml 파일을 작성해요

    • 🥽 Netlify.toml 파일 작성 방법


  • 자신의 프로젝트를 Github 에 푸쉬해요

    • 🔌 Github 사용 방법


  • Netlify 에 접속하여, New site from Git 를 클릭해요

    • 이 속성은 Netlify의 홈(처음) 페이지에서 하단으로 스크롤하면 있어요!


  • 그다음, GitHub를 선택하고 업로드할 저장소를 선택해줘요

    • 아까 푸쉬한 리포지토리를 업로드 해줘요!


    여기서 가장 중요한 과정은, 첫번째 과정이라고 생각되요😎

    Netlify.toml 파일을 작성하는 방법을 정리한 글을 꼼꼼이 읽어주세요! 😉
