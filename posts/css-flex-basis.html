flex 사용 방법

flex는 display: flex;로 사용할 수 있어요

flex 의 flex-container, flex-item 개념

flex-container는 flex의 영향을 받는 전체 공간이에요

그리고, flex-container의 요소들을 flex-item이라고 해요 🤟

flex 의 축 개념

flex-item이 배치된 방향이 메인축이고, 메인축의 방향의 반대 방향이 교차축이에요 😄
