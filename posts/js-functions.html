padEnd

형식

str.padEnd(만들고자하는 문자열 길이, 만들고자하는 문자열)

예제

                    var str = 'Hello';

                    str.padEnd(5, '.'); // Hello.....

split

형식

str.split(끊어야할 부분을 나타내는 문자열, 끊어질 문자열의 최대 개수)

예제

                    var str = 'hello world';
                    str.split(' ') // ['hello', 'world']

join

형식

array.join(배열의 각 요소를 구분할 문자열)

예제

                    var arr = ['a', 'b', 'c'];

                    arr.join(''); // abc

map

배열을 특정 함수를 사용해서 변환하고 싶을때 사용하는 함수에요!

형식

예제

                    const numbers = [1, 2, 3, 4, 5];
                     
                    const squared = number.map(n => n * n); // n은 number의 값들을 의미
