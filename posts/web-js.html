JavaScript 기본기
2021년 8월 17일 화요일 ☁️

JavaScript 는 페이지를 동적으로 움직일 수 있게 하는 프로그래밍 언어에요


JavaScript 는 HTML 에서 <script> 를 선언하여 사용해요

주석

// ... 로 선언해요

; (세미콜론)

JavaScript 는 모든 코드가 끝나는 끝에 ; 을 붙어요. 이 기호는 세미콜론 이라고 읽어요

변수

var Name = Value; 로 선언해요

상수

let Name = Value; 로 선언해요

const Name = Value; 로 선언해요

함수

함수는 function() { ... }; 로 선언해요 
                    function() {
                        id.innerText = "hello, World!";

                    };
