Netlify를 사용하여 배포할때 CSS 작성 팁
2021년 7월 27일 화요일 🌤

저가 Netlify로 배포를 할때 CSS에서 설정해놓은 Height속성이 20vh로 설정 되어있었어요

Netlfiy에 올려진 페이지를 보니까 약 1vh가 올려저 설정되어 있더라고요.. 😭

그래서, 1vh를 낮추니 Netlify로 올려진 페이지와 , 기존에 file 형태로 확인한 페이지와 동일하게 보이더라고요! 🤩

CSS의 Size 관련 속성이 모두 다 이런지는 모르겠지만 , Netlify에 배포한 페이지와 file로 확인한 페이지가 다르다면 , 이 방법을 확인해보는 것도 나쁘지 않은거 같아요!
