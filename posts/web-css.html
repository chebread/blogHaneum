CSS 기본기
2021년 8월 16일 월요일 ☀️

CSS 의 기본 문법과 활용 예제를 통해 CSS 를 더 배워바요!

HTML 에서의 style Tag

HTML 엔 ﹤style﹥ 태그가 있어요

이 Tag 는 CSS 를 사용할때 CSS 코드를 작성하기 위한 공간이에요


만약, CSS 코드가 너무 많아서 HTML 코드와 분리하고 싶다면, 이런식으로 코드를 작성하면 되요!

HTML 파일

                    <head>
                        <style> <!-HTML 에서 Style Tag 를 삭제해요-->
                            ...
                        </style> 
                    </head>

CSS 파일

                    <style> <!-CSS 파일을 한개 만들어준후, CSS 파일에 style Tag 는 삭제한체로 복사해줘요-->
                        ...
                        ...
                        ...
                    </style>

CSS 기본 문법

CSS 는 정해진 문법을 통해 HTML Tag 들을 제어해요

이제 Tag 를 제어할 수 있는 CSS 문법을 예제로 배워볼게요!

페이지의 배경 색상을 바꾸는 방법

                    body {
                        background-color: Colors;
                    }

Colors 에 정하고 싶은 색상을 정의하면 되요!

제목의 폰트 크기를 바꾸는 방법

                    .title {
                            font-size: Xpx;
                    }

X 에 정하고 싶은 크기를 설정하면 되고, 단위는 px, em, vw, ... 사용할 수 있어요

이 단위들은 정리해놓은 글을 확인해주세요! (CSS 단위 정리)
