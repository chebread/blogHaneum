Firebase 기본기
2021년 10월 30일 토요일 ⛅️

이 블로그는 React Hooks를 이해한다는 전제하예 작성되었어요

만약 React Hooks를 잘 모르겠다면, 이 블로그를 읽어보세요!

React 기본 설정

1. npx create-react-app (e.g: bloghaneum)

먼저, Terminal에서 npx create-react-app 으로 React App 을 만들어요

여기서 주의 해야할 점은, 반드시 소문자로 React App 을 만들어야한다는 거에요!

2. npm install firebase

그리고, 자신이 만든 React App 디렉토리에 접근하여 npm install firebase 을 입력하여
Firebase Module 들을 설치해요

3. CRA (create-react-app) 로 만들어진 파일중에서 불필요한 파일을 삭제해요

React App 의 홈디렉터리 기준으로, 지워야할 파일들과 디렉토리를 적어봤어요!
