2021년 8월 24일 화요일 🌧

Tag

JSX에서는 Tag를 열었으면 꼭 닫혀있어야 해요!

                    import React, { Component } from 'react';

                    class App extends Component {
                        render() {
                            return (
                                <div>
                                    <input type="text">
                                </div>
                            );
                        }
                    }

                    export default App;

엘리먼트

두개 이상의 엘리먼트(Tag)는 무조건 한개의 엘리먼트로 감싸져있어야 해요! 
                    import React, { Component } from 'react';

                    class App extends Component {
                        render() {
                            return (
                                <div> {/*이렇게!*/}
                                    <div>
                                        hello!
                                    </div>

                                    <div>
                                        hello
                                    </div>
                                </div>
                            );
                        }
                    }

                    export default App;

JavaScript의 varible을 사용하는 방법

콧수염 괄호안에 변수를 적어 사용하면 되요!

                    hello { varible }

삼향연산자

삼향연산자는 if 문을 사용할때 사용해요

                    {
                        1+1 === 2
                            ? (True section)
                            : (False section)
                    }

AND 연산자

true일때 발생할 이벤트는 있지만 false 일때는 아무것도 없는 연산자에요

                    1 + 1 === 2 && (True)

className

JSX에서 class를 사용하고 싶으면, class의 구문을 className으로 변경하면 되요!

                    const style = { {/*객체로 style를 작성해야 되요*/}
                        backgroundColor: 'black',
                    }

                    return <style={style}>Hello</style>;
